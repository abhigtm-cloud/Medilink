### **Step 1:** Go to Render
→ https://render.com

### **Step 2:** Click "New +" → "Web Service"
→ Select "Build and deploy from Git repository"

### **Step 3:** Connect & Select Repository
→ Authorize GitHub
→ Select: **abhigtm-cloud/Medilink**

### **Step 4:** Configure Service
- **Name:** `medilink-app`
- **Environment:** Docker
- **Region:** Oregon
- **Plan:** Free (or Starter for better performance)
- **Branch:** main

### **Step 5:** Deploy!
→ Click "Create Web Service"
→ Wait 5-10 minutes for build
→ Your app will be live at: **https://medilink-app.onrender.com**
